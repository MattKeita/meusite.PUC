Sobre o trabalho

Este projeto demonstra organização de arquivos (css/, js/, images/), uso de HTML sem frameworks, estilos reaproveitáveis em classes e validação de formulário.

O que mostrar no vídeo

  1. Apareça e diga seu nome.
  2. Mostre o site local (index.html) e navegue pelo menu.
  3. Abra o formulário, mostre o código de validação (formHandler.js) e faça um envio com GET.
  4. Mostre formAction.html processando os dados com Javascript.
  5. Comente o uso de classes CSS reaproveitáveis e a organização por pastas.
